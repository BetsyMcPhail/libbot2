Detailed Description

Batching OpenGL commands for delayed rendering.

bot_vis/bot_vis.h

Batched GL drawing allows a program to issue some basic OpenGL drawing commands while an OpenGL context is not active. Commands will be queued up, stored, and then executed when requested via bot_bgl_render().
