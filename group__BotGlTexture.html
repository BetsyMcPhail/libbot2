Rendering images/textures.

Convenience class for uploading and rendering textures in OpenGL.

Linking: pkg-config --libs bot2-vis

◆ bot_gl_texture_draw()

void bot_gl_texture_draw ( BotGlTexture *  t)

Renders the texture in a unit square from (0, 0) to (1, 1). Texture coordinate mapping:

texture          opengl
0, 0          -> 0, 0
width, 0      -> 1, 0
0, height     -> 0, 1
width, height -> 1, 1

all opengl Z coordinates are 0.

Parameters

typeOpenGL data type used to represent the texture pixels. Supported types are: GL_UNSIGNED_BYTE, GL_BYTE, GL_UNSIGNED_SHORT, GL_SHORT, GL_UNSIGNED_INT, GL_INT, GL_FLOAT.
stridenumber of bytes separating the start of each row in the texture data.

Uploads texture data to OpenGL-managed memory. The original data is not needed for rendering after an upload.
