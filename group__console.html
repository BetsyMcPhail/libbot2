Detailed Description

Vertically scrolling text.

bot_vis/bot_vis.h

Utility class for rendering vertically scrolling text. Text can be added with printf-style formatting, and will scroll vertically as more text is added. A decay value can also be set to fade away text over time.

This class can be useful for displaying textual debugging or status information in an OpenGL window.

Linking: pkg-config --libs bot2-vis
