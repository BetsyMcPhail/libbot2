Detailed Description

quick and dirty profiling tool.

bot_core/bot_core.h

inspired by the matlab tic/toc command

call bot_tictoc("description") to set the timer going call it again with the same description to stop the timer

Note: To get output, set the "BOT_TICTOC" environment variable to something
